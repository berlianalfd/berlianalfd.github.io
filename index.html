Saya seorang fresh graduate D4 Teknik Elektronika Politeknik Negeri Bandung dengan ketertarikan dalam pengembangan teknologi, sistem berbasis mikrokontroler, dan otomatisasi.

Selama kuliah, saya aktif mengerjakan proyek berbasis IoT, mengembangkan perangkat dengan sensor GSR dan logika fuzzy, serta membuat aplikasi mobile untuk monitoring terapi. Saya juga pernah magang di perusahaan teknologi dan terlibat langsung dalam desain serta implementasi sistem elektronik.

Saya antusias dalam memecahkan masalah teknis, bekerja dalam tim, dan selalu ingin belajar hal baru di dunia teknologi!
